Key Features

Local Storage

All passwords are stored locally on your device, never uploaded to any server.

Hidden Passwords

Passwords are never shown on screen, they can only be copied to clipboard.

Quick Copy

Copy passwords directly to clipboard with a single click. Clipboard clears automatically.
